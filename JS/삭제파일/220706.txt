제이쿼리 사용
 $ ( 선택자 ).동작함수();

 // $는 jquery () 함수의 축약 형태로 사용

값 가져오기 & 설정하기
- $(선택자).val();
 input 

Style 변경하기
-> $(선택자).css("속성", "속성값");

text 변경하기
-> $(선택자).text(글자);

속성 변경하기
 -> $(선택자).attr("속성", "속성값");

HTML 변경하기
 -> $(선택자).html(html);

요소 추가

-> 선택 된 요소의 자식 요소 중 마지막으로 추가
-> $("선택자").oppend(요소);

요소 탐색
-> $(선택자).prarend().함수()
 선택된 요소의 부모 요소 선택
-> $(선택자).next().함수()
 선택된 요소 바로 다음에 위치한 형제 요소 선택
-> $(선택자).children().함수()
 선택된 요소의 자식요소 모두 선택


클래스 조작하기

$(선택자).addClass
- 선택된 요소에 클래스 추가
