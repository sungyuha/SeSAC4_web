반응형 웹
 - 디스플레이 종류에 따라 화면의 크기가 자동으로 최적화되는 웹 페이지

적응형 웹
 - Pc용 사이트와 모바일 용 사이트를 별도로 제공하는 웹 사이트

CSS 단위

 px : 화면의 하나의 점
 - 반응형에 부적합

 % : 상위 요소의 상대적인 크기

 em : 주로 폰트에서 적용

 em주의하기 : 굳이 필요한 상황이 아니라면 자주 사용하지 않음

 rem : 최상위 요소 (html) 크기에 비례함
 - 16px * 2 = 32px

viewport 단위
 - 웹 브라우저에서 웹 페이지가 표시되는 영역
 - 탭 부분을 제외하고의 크기
