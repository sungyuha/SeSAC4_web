map() 함수 : 리액트에서 자주 사용하는 반복문

map () 함수 문법
 callbackFunction 
  - 새로운 배열의 요소를 생성하는 함수로, currentValue, index, array 3개의 인수를 가질 수 있다.
  - [this.Arg] 는 생략 가능한 것으로 callbackFunction 에서 사용 가능한 this 객체

자바스크립트 베열 객체의 내장 함수인 map 함수를 사용,
map 함수는 피라미터로 전달 된 함수를 사용해 배열 내 각 요소를 원하는 규칙에 따라 변환 한 후 그 결과로 새로운 배열을 생성


filter() 함수
- 배열의 특정 항목을 지울 때 사용
- 배열에서 특정 조건을 만족하는 원소들만 쉽게 분류할 수 있음
