배열 구조 분해
 const [변수] = 배열;
- 각 변수에 배열의 인덱스 순으로 값 대응

객체 구조 분해
 const {변수} = 객체;
- 객체 안의 속성을 변수명으로 사용

콜백 함수
 - Callback : 함수가 끝난 뒤 실행되는 함수
 - 함수를 만들 때 parameter로 함수를 받아서 사용한다.

콜백 지옥 ( Callback Hell )
- 비동기 프로그램 시 발생하는 문제
- 함수의 매개변수로 넘겨지는 콜백 함수가 반복되어 코드의 들여쓰기가 너무 많다.. -> 안좋은 코드

프로미스(Promise)
- 
- 비동기 작업에서 성공과 실패를 분리해서 메소드를 수행한다.
- resolve(value) : 일이 성공적으로 끝난 경우

async/await
- 프로미스 기반 코드를 좀 더 쓰기 쉽고 이해하기 위해 만들어진 것
- 비동기 코드를 동기식으로 작성 가능하게 해준다.
- 자바스크립트의 비동기 처리 패턴 중 가장 최근에 나온 문법

async
- 함수 앞에 붙여 Promise를 반환한다.
- 프로미스가 아닌 값을 반환해도 프로미스로 감싸서 반환한다.

await
- '기다리다' 라는 뜻을 가진 영단어
- 프로미스 앞에 붙여 프로미스가 다 처리될 때까지 기다리는 역할을 하며 결과는 그 후에 반환한다.
