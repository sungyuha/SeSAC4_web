npm
. 노드 패캐지를 관리해주는 틀

npm 사용하기
npm list

npm init --yes
- package.json생성 된다.
npm install 패키지 이름

method(메소드)
get : 요청을 받을 떄 ex)검색할때, 서버에 정보를 요청할때 많이 사용
post : 어떤 것을 만들때, ex) 회원가입 시 정보가 안보임(아이디 같은 정보로 검색할때 사용도 함)
put : 수정할때
patch : 수정할때 / 특정하게 보낸것만 업데이트
delete : 삭제할때
