미들웨어
- 클라이언트와 서버가 요청과 응답을 할 때 중간에서 거쳐가는 함수
- 미들웨어에서는 request, response 객체를 모두 사용 가능하고 next() 함수를 이용해 다음 미들웨어로 접근이 가능하다.

파일 업로드
- 클라이언트 -> 서버 데이터 전송하는 법

multer
- 파일 업로드를 위해 사용되는 미들웨어
none() : 파일을 업로드 하지 않겠다

multer 세부 경로
- 경로뿐 아니라 파일명 등을 직접 지정, 제어하고 싶을 때

.stotage : 저장할 공간에 대한 정보
.diskStorage : 하드디스크에 업로드 파일을 저장
.destination: 저장할 경로
.filename : 파일명
.limits : 파일 제한
.fileSize: 파일 사이즈 제한
