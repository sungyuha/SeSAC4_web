비동기 http 통신 : 폼의 데이터를 서버서와 dynamic하게 송신 하는 것
dynamic : 웹 문서가 정적으로 멈춰있는 것이 아니라 일부 내용이 실시간으로 변경 되는 것

비동기 HTTP 통신 방법
1. Ajax : 비동기 자바스크립트 / 자바스크립트를 이용해 클라이언트와 서버 간에 데이터를 주고 받는 비동기 HTTP 통신
 -> 실무에서는 많이 사용하지 않음 / 그러나 제이쿼리를 이용해서 사용하면 쉬워짐
2. Axios : node.js와 브라우저를 위한 Promise API를 활용
3. Fetch : ES6 부터 들어온 JavaScript 내장 라이브러리 / 최신 문법이라 브라우저 호환성이 낮다
